# Project 04 — Regulatory Change Management

## Overview
A complete Regulatory Change Management program for FinSecure Pvt Ltd
covering tracking, impact assessment, and SLA management of regulatory
changes from RBI, MeitY, FIU-IND, and SEBI. All changes are 2026 current.

## Files
- `FinSecure-Regulatory-Change-Register-v1.0.xlsx`

## Tabs Inside the Excel
- **Change Register** — 10 regulatory changes tracked end to end
- **Impact Assessment** — worked example for RBI AI Guidelines 2026
- **SLA Dashboard** — RAG status · upcoming deadlines · regulator summary
