# GRC Portfolio — FinSecure Pvt Ltd

A end-to-end GRC program built for a fictional 
NBFC (FinSecure Pvt Ltd) demonstrating hands-on 
knowledge of risk management, compliance frameworks, 
and information security controls.

## Frameworks Covered
- ISO 27001
- SOC 2 Type II
- DPDP Act 2023
- RBI NBFC Master Directions
- PCI-DSS
